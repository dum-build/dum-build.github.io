For web projects, knowledge of JavaScript is not necessary.

simple

It's under 1000 lines of code and does not use plugins or build scripts.

It just applies jstransformers list based on file extensions. Files with no associated transformer will be copied directly.

easy

Your package.json a file listing npm dependencies is your configuration.

scripts: { serve: dum serve },
devDependencies: {
	dum: dum-build/dum,
	jstransformer-jade: ,
	jstransformer-stylus: 
}
